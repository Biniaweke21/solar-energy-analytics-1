# Solar Strategy Optimizer 
 
An analytics engineering project to identify high-potential solar investment regions in Benin, Sierra Leone, and Togo. 
 
## Environment Setup 
 
1. **Clone the repository:** 
   ```bash 
   git clone <your-repo-url> 
   cd Solar-Strategy-Optimizer 
   ``` 
 
2. **Create a virtual environment:** 
   ```bash 
   python -m venv senv 
   ``` 
 
3. **Activate the environment:** 
   - Windows: `senv\Scripts\activate` 
   - macOS/Linux: `source senv/bin/activate` 
 
4. **Install dependencies:** 
   ```bash 
   pip install -r requirements.txt 
   ```
